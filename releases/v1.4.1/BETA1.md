# DoubleDoors v1.4.1 BETA.1 Release Notes

Release date: 2026-05-06

## WARNING

This is a BETA release intended for testing purposes only. It may contain bugs or incomplete features. Do not use this version on production servers.

## Highlights

- Added multiversion support, bringing back compatibility with Minecraft 1.21.X while keeping the current 26.1.X support.

## Added

- Added multiversion support for Minecraft 1.21.X, allowing users to run DoubleDoors on both 1.21.X and 26.1.X servers with the same plugin version.
- Added tests to ensure compatibility across supported Minecraft versions.

## Artifacts

- Bukkit/Spigot/Paper/Purpur: doubledoors-bukkit-1.4.1-beta.1.jar
- Optional proxy companion: doubledoors-proxy-1.4.1-beta.1.jar

## Notes

This BETA release is focused on testing the new multiversion support. Please report any issues encountered during testing to help us improve the final release.
