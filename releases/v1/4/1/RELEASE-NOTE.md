# DoubleDoors v1.4.1 Release Notes

Release date: 2026-05-07

## Highlights

- Added multiversion support, bringing back compatibility with Minecraft 1.21.X while keeping the current 26.1.X support.

## Added

- Added multiversion support for Minecraft 1.21.X, allowing users to run DoubleDoors on both 1.21.X and 26.1.X servers with the same plugin version.
- Added tests to ensure compatibility across supported Minecraft versions - this will future-proof the plugin for new versions.

## Artifacts

- Bukkit/Spigot/Paper/Purpur: doubledoors-bukkit-1.4.1.jar
- Optional proxy companion: doubledoors-proxy-1.4.1.jar

## Notes

This release focuses on providing multiversion support for Minecraft 1.21.X while maintaining compatibility with 26.1.X. Users can now run DoubleDoors on both versions without needing separate plugin versions. Please report any issues encountered to help us continue improving the plugin.
